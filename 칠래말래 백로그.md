- **상황**: 무료 키 한도가 콘솔상 5,000/일 → 실제 **1,000/일**로 확인됨. 백필 중 한도 초과로 ODSay 응답 차단됨.
- **영향 범위**:
  - 새 단지 backfill 불가 → `transit_path_cache`에 없는 (단지 × CBD) 페어는 RouteOptions/SubwayPathDisplay 카드 깨짐
  - Vercel 운영은 캐시 히트만 정상, 캐시 미스 시 fallback 매트릭스로 떨어짐
- **단기 조치**:
  - [ ] **자정(KST 00:00) 한도 리셋 대기** — 다음 회차 backfill 재개
  - [ ] **Wave 1 우선순위 유지** — 한도 회복 시 신혼 핵심 8구·6신도시·송도부터 (`PRIORITY_DISTRICTS`)
  - [ ] **일일 안전 limit 900 엄수** — `backfill-transit-cache.mjs` 기본값 검증
- **중기 조치**:
  - [ ] **ODSay 유료 플랜 검토** — Wave 1 18일 + Wave 2 12일 + Wave 3 38일 = ~68일 무료 → 유료로 단축
